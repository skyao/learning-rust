常量是绑定到一个名称的不允许改变的值。

常量与不可变变量的区别：

- 不允许对常量使用 `mut`
- 声明常量使用 `const` 关键字而不是 `let`，并且 *必须* 注明值的类型。
- 常量只能被设置为常量表达式，而不能是函数调用的结果，或任何其他只能在运行时计算出的值

备注：rust的常量是必须在编译时明确赋值的。

声明常量的例子：

```rust
const MAX_POINTS: u32 = 100_000;
```
